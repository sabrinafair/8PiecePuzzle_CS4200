1. create folder to pull repository code into
2. clone repo

3. open IDE you would like to use, these directions use <i>Eclipse</i>
4. create a new project
<br><i>note: finish, you don't need a module</i><br>
6. create a new package named `puzzleSolver`
7. right click package and select `import`
8. General -> File System -> Browse
<br>`> import .java files`
9. run the main class
6. select option given from menu
